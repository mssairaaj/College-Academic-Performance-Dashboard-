# College Academic Performance Dashboard

## Project Overview
This project analyzes student academic data and builds an interactive dashboard using Microsoft Excel. The objective is to monitor student performance, attendance patterns, and departmental statistics to support data-driven decision making in academic institutions.

The dashboard provides a centralized view of key metrics such as attendance rates, CGPA distribution, department-wise performance, and placement statistics.

## Tools and Technologies
- Microsoft Excel
- Power Pivot
- Pivot Tables
- Data Modeling
- Dashboard Visualization
- Data Cleaning and Validation

## Dataset
The dataset used in this project includes information related to:

- Student details
- Subjects and exam results
- Attendance records
- Department information
- Placement and hiring statistics

The data was cleaned and structured to ensure accurate analysis and reporting.

## Key Features
- Built a data model using **Power Pivot** to establish relationships between multiple datasets.
- Used **Pivot Tables** to summarize and analyze academic performance metrics.
- Created **interactive dashboards with charts and slicers** for dynamic data filtering.
- Performed **data cleaning and validation** to improve data quality.
- Conducted **Exploratory Data Analysis (EDA)** to identify patterns in student performance and placement outcomes.

## Insights Generated
- Department-wise student performance comparison
- CGPA distribution analysis
- Attendance trend analysis
- Placement hiring trends
- Overall academic performance monitoring

## Project Outcome
The dashboard provides a centralized and interactive view of academic performance metrics, enabling users to easily monitor student performance and gain insights through data visualization.
